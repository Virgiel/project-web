The project

This is a two weeks project at EFREI for the web programming course. The goal of this course and project is to learn the basic of web development (HTML, CSS & JS) before learning web frameworks. It's the reason why this website is 100% handmade. The project theme was free but it had to enable user interaction and dynamic content. We have chosen to do simulation because they are both fun to create and to plays with.

We hope you find this website interesting and invite you to discover our work.
